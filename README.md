# AI-Content-Generator
This project is an AI-powered content generation tool that creates high-quality blog posts and social media captions based on user input. Users can select a topic, tone, and content type, and the application generates relevant content while displaying word and character counts.
## Overview

AI Content Generator is a simple web application built using Python and Streamlit. It helps users generate blog posts and social media captions based on a selected topic and tone. The application also displays word count and character count for the generated content.

## Features

- Generate Blog Posts
- Generate Social Media Captions
- Multiple Tone Options
  - Professional
  - Friendly
  - Creative
  - Motivational
- Word Count Display
- Character Count Display
- Regenerate Content Option
- User-Friendly Interface

## Technologies Used

- Python
- Streamlit

## Installation
Clone the repository:

```bash
git clone https://github.com/your-username/AI-Content-Generator.git
```

Move into the project directory:

```bash
cd AI-Content-Generator
```

Install required packages:

```bash
pip install -r requirements.txt
```

## Running the Application

Run the Streamlit app:

```bash
streamlit run app.py
```

The application will open in your browser automatically.

## How to Use

1. Enter a topic.
2. Select a tone.
3. Choose content type:
   - Blog Post
   - Social Media Caption
4. Click on **Generate Content**.
5. View the generated content.
6. Check word count and character count.
